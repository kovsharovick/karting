# User stories

> Этап 2. Пользовательские истории. Формат: «Как <роль>, я хочу <действие>, чтобы <ценность>».
>
> **Скоуп: клиентское приложение и API для него** (роль «Клиент»).

| ID | Как (роль) | Хочу (действие) | Чтобы (ценность) | Критерии приёмки | FR |
| :-- | :-- | :-- | :-- | :-- | :-- |
| US-1 | Клиент | войти в приложение по номеру телефона и одноразовому SMS-коду без пароля | быстро начать пользоваться без сложной регистрации | Ввожу телефон, получаю SMS-код, подтверждаю вход; пароль не требуется. | FR-1 |
| US-2 | Клиент | видеть список слотов заездов на ближайшие 7 дней | выбрать подходящий заезд | По умолчанию показываются слоты на 7 дней (R-027); при отсутствии слотов — «Пока нет доступных заездов»; более длинный период доступен через фильтр дат. | FR-2 |
| US-3 | Клиент | фильтровать слоты по дате, конфигурации трассы, наличию мест и маршалу | быстрее найти подходящий вариант | Фильтры применяются к списку; результат обновляется по выбору. | FR-3 |
| US-4 | Клиент | открыть карточку слота с полными параметрами | понять условия заезда до записи | Вижу дату/время, конфигурацию трассы, маршала и его средний рейтинг, всего/свободно мест, предупреждение по возрасту/росту, итоговую стоимость, адрес и место сбора. | FR-4, FR-5, FR-13 |
| US-5 | Клиент | записаться на выбранный слот | забронировать место на заезде | При наличии мест создаётся бронь; запись появляется в «Моих бронированиях». | FR-6 |
| US-6 | Клиент | одной бронью записать себя и нескольких гостей | не оформлять каждого отдельно | Указываю число мест; персональные данные гостей не запрашиваются. | FR-7 |
| US-7 | Клиент | для каждого участника выбрать свою или прокатную экипировку | приехать со своим шлемом/подшлемником или взять прокат | «Своя» не расходует прокатный фонд; «Прокат» — только при наличии свободного фонда из API. | FR-8, FR-10 |
| US-8 | Клиент | быть защищённым от записи сверх лимита | не приехать на переполненный заезд | Число мест ограничено `min(свободные карты, потолок конфигурации трассы)`; при отказе API бронь не создаётся, показывается актуальная доступность. | FR-9, FR-11 |
| US-9 | Клиент | видеть список своих бронирований | контролировать предстоящие и прошедшие заезды | Вижу свои брони со статусом (активна / отменена клиентом / отменена центром / завершена), параметрами слота, числом мест и экипировкой. | FR-14 |
| US-10 | Клиент | отменить свою запись до старта | освободить место, если планы изменились | До 1 ч до старта — свободная отмена; менее чем за 1 ч — отмена с предупреждением, без штрафов; поздняя отмена учитывается в статистике центра. | FR-15, FR-16, FR-17 |
| US-11 | Клиент | видеть итоговую стоимость заезда и проката | понимать сумму заранее | Цена показана из API; оплата офлайн (наличные / перевод), онлайн-оплаты в приложении нет. | FR-12 |
| US-12 | Клиент | после завершённого заезда оценить маршала | поделиться обратной связью и видеть рейтинг других | Оценка необязательна, однократная, без редактирования; средний рейтинг маршала виден в карточке слота. | FR-19, FR-5 |
| US-13 | Клиент | получать push-напоминание перед заездом | не пропустить время старта | Перед стартом приходит push-напоминание. *(Should)* | FR-20 |
| US-14 | Клиент | получать push при отмене заезда центром | узнать об отмене и причине вовремя | Бронь остаётся в списке со статусом «Отменён центром» и причиной; приходит push; повторная запись на этот слот недоступна. | FR-18, FR-21 |
