## 1. TC-AUTH-001: Запрос OTP-кода с индивидуальным таймером

| Поле | Значение |
|------|----------|
| **ID** | TC-AUTH-001 |
| **Приоритет** | P0 |
| **Предусловия** | Приложение запущено, страница `/login` открыта |
| **Шаги** | 1. Ввести телефон +7 (999) 111-11-11, нажать «Получить код»<br>2. Дождаться перехода на экран ввода кода<br>3. Вернуться назад и сменить номер на +7 (999) 222-22-22<br>4. Нажать «Получить код» |
| **Ожидаемый результат** | • Для второго номера отправка доступна сразу<br>• Таймер для первого номера не блокирует второй<br>• В консоли приложения появляется лог `OTP for +79991111111: XXXX` |
| **Мок-данные** | `AuthService.requestCode()` генерирует код и сохраняет в БД `otp_codes` |
